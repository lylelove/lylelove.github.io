技术

1.whoogle-search [1]

一个自托管，无广告，尊重隐私的元搜索引擎。获取Google搜索结果，但不包含任何广告，javascript，AMP链接，cookie或IP地址跟踪。一键式轻松部署为Docker应用程序，并且可通过单个配置文件进行自定义。快速简便地实现为台式机和移动设备上的主要搜索引擎替代品。

2.CSS Inspiration[2]

这里可以让你寻找到使用或者是学习 CSS 的灵感，以分类的形式，展示不同 CSS 属性或者不同的课题使用 CSS 来解决的各种方法。

3.Obsidian + Seafile搭建自己的私有笔记，附使用入门[3]

Obsidian 是一款跨平台知识管理软件，支持Windows、MacOS、Linux系统，最近还增加了移动端（Android、IOS）支持，官方承诺个人用户免费。Obsidian使用Markdown编辑器，如果您已经使用过Markdown，那Obsidian对你来说应该比较顺手。

4.在 Visual Studio Code 中构建 Java 应用程序 [4]

多年来，Java的发展一直由三个主要的 IDE 主导：Eclipse、IntelliJ IDEA 和 NetBeans。但我们还有其他合适的可选项。在众多的多语种代码编辑中，VScode已被证明是一款出色的产品，并支持java。VS Code 还为其他技术堆栈提供世界一流的支持，包括前端 JavaScript 框架、节点.js 和 Python。

5.你可能不知道的 css 动画性能[5]

你一定知道 JS 动画的优先级 < css 动画。即使必要，用 JS 操作 class 的优先级也一定 > 用 JS 直接修改具体样式。

但是如果问到：“你了解 css 动画的性能么？如何优化？”你该怎么解决？

6.1loc [6]

如果你想知道一行 JavaScript 代码能实现哪些比较实用的功能，可以看看这个 GitHub项目：1loc。该项目收集了共计 126 个代码片段，其中包含对数组、日期、函数、DOM 等常见代码模块的组件实现。

有趣

1.sparrow[7]

场景化低代码（LowCode）搭建工作台，实时输出源代码。

2.macos-virtualbox[8]

帮你在 VirtualBox 上安装 macOS 操作系统的工具。这个工具是一个 Bash 脚本，运行后只需要按回车即可完成安装。目前已支持 Linux 、Windows、macOS 多个主流操作系统，可安装 Catalina (10.15)、Mojave (10.14) 和 High Sierra (10.13) 等多个不同版本的苹果系统

3.Windows11 in React[9]

这个开源项目旨在使用 React、CSS (SCSS) 和 Js 等标准 Web 技术在 Web 上复制 Windows 11 桌面体验。让您体验win11的桌面UI等细节。

笔者在刚刚放出dev渠道时就更新了win11，两个多月的体验下来，从来没有遇见过Bug，从网页小游戏到荒野大镖客都能完美运行，JetBrains家的IDE也正常使用，完全可以作为主力机使用了。当然，最重要的是，win11比win10真的好看太多了！

4.YesPlayMusic[10]

高颜值的第三方网易云播放器，支持 Windows / macOS / Linux

5.MisakaTranslator [11]

御坂翻译器—Galgame/文字游戏/漫画多语种实时机翻工具

6.LifeAt Virtual Spaces[12]

有时候真的需要这样的工具来让自己静下来！网站可以在线营造专注的学习和工作空间，舒缓的音乐加上唯美的画面，看着看着你就睡着了。

资源

1.摩根图书馆与博物馆[13]

建于1906年，最初为私人图书馆，1924年开始对公众开放。博物馆主要收藏各类古籍珍本、文学艺术历史手稿和艺术品，重要藏品包括3本《古登堡圣经》、弥尔顿《失乐园》唯一现存的手稿、美国《独立宣言》等。数字资源开放获取，可下载高清图。

2.YYDS电影[14]

一个电影电视剧下载网站，提供阿里云、百度云等大网盘下载链接。

3.1G 数据、70+软件、万字长文，手把手教你如何将 Win10 从零打造成工作『利器』[15]

作者从系统优化开始，洋洋洒洒写了一万多字，包含了70多个软件的使用和安装，是购买新电脑或重装系统后的参照指南。

4.关于收入和财富不平等[22]

wid.world 是个不错的网站，里面有很详细的各国收入不平等的数据。上图分别是美国的 top 0.1%, top 1%, bottom 50% 百分位收入的占比（截止到2019年）。可以很明显的看到，自从1980年左右开始，收入不平等就在一路狂飙：收入高的人的收入占比在持续升高，收入低的人的收入占比在持续下降。现在的不平等程度已经差不多回到了1930年代的水平了。
