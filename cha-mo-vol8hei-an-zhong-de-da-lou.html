技术

1.geoblender

GitHub 上的一个开源教程。主要教你如何使用 QGIS 和 Blender 软件，制作带有阴影效果的浮雕地图。

2. github-FE-project

GitHub 值得收藏的前端项目。

3.PathPlanning

动画算法库：PathPlanning，主要实现多种在机器人领域中，常用到的路径规划算法，其中也包括基于搜索与采样的方法。

为了让开发者便于理解，作者特意为每种算法设计了动画，用以展示路径规划算法的运行过程。

4. pythontutor.

代码可视化，可将代码按执行步骤，分段可视化展示，并允许将可视化代码内嵌到网站上。

目前已支持 Python、Java、C、C++、JavaScript、Ruby 等编程语言，该工具可应用到文档教程、技术讲座、团队新手指引项目等场景上。

5. Flipper

Facebook 在GitHub上开源的一款移动端调试工具，可在 PC 桌面上可视化调试与控制 iOS、Android 与 React Native 应用工具。

6. 代码审查的完整指南

代码审查是软件开发团队广泛接受的最佳实践。在这篇文章中，我们将介绍为什么最成功的团队使用代码审查，如何在您的开发过程中采用代码审查，以及什么是最佳实践。

8. RawGraphs

一款比较实用的数据可视化工具，通过粘贴或上传表格数据，即可快速生成数据可视化图形。

表格数据支持 CSV、TSV、Excel 等格式，图形提供瀑布图、条形图、饼状图、散点图等多种样式。

另外，开发者还可直接在页面上对数据进行可交互式的比对分析，完成后即可将数据导出成 PNG 与 SVG 格式图片，方便进行后续加工编辑。

10. Microsoft Office 365搭建邮局开启外部邮箱转发

11.《观妙中国》新专题：首个司马台长城的 360 度实景游览

观妙中国是一款线上艺术品与历史文物展示应用，支持 iPhone 与 Android，汇聚了中国超过30 家博物馆及艺术文化机构的典藏精品。前几日发布了新专题：见微知“筑”识长城，可以让用户在手机上以 360 度无死角的方式，虚拟参观司马台长城。

一些闲话

昨天晚上大约十一点半光景，孙老师突然发微信问我睡没睡，有没有空帮他的研究生写点代码。

我以为是多么复杂的算法，结果老师说只是导入数据到matlab里。

我一直以为我们管理科学与工程这一大类的研究生都应该几种常用语言能用个大概，特别还是搞算法必须用到的matlab，至少不是连数据导入这种事情都不会吧，然而好像事实就是如此。

有空就多学点吧，别大晚上凌晨了还把学弟摇起来帮忙。
